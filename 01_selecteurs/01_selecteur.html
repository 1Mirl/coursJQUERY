JQUERY- Introduction

JQUERY: write less... do more

JQUERY est une API/ 1er h2

Paragraphe 1

JQuery est ce qu'on appelle une bibliothèque ou une API (application programming interface) Javascript, c'est à dire un ensemble de fonctions qui permettent d'écrire de manière condensée, les lignes de codes les plus utilisées en Javascript.

La version 1.0 stable de JQuery a été crée en aout 2006 par un jeune homme de 22 ans du nom de John RESIG. Aujourd’hui nous en sommes à : JQuery JavaScript Library v3.5.1

On peut trouver du jQuery ici

JQUERY écrit d'une maniére plus condensée

Nous allons souvent utiliser les fonctions Javascript qui permettent d'attraper les balises HTML et les animer, ainsi en Javascript nous allons écrire d'une certaine manière et en JQuery d'une manière plus condensée

jQuery est une bibliothèque JavaScript créée par John Resig en 2006 afin de faciliter la lecture et l’écriture du JavaScript. Une vingtaine de lignes de code en JavaScript correspond à quelques lignes seulement en jQuery. D’où le slogan : "Write less… Do more".

Exemple :

  • Javascript :document.getElementsByTagName('p')(0).innerHTML = 'Je change le contenu du 1er p de tous les p avec JS.';
  • jQuery :$('p:first').text('Et là je change le texte avec jQuery');

Paragraphe 2

  1. la fonction jquery $() est l'équivalent d'un document.querySelectorAll()
  2. la méthode eq() permet de sélectionner en jQuery un indice ou une collection d'objet
  3. la méthode html() remplace innertHTML pour récupérer ou redéfinir un contenu d'élément
  4. La méthode .css() applique des changements en css en remplacement de style()
